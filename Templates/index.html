About our product

Artificial Intelligence and Machine learning have multiple applications in medical science, technology, and our day-to-day lives. In the past decade, information technology has seen a huge uprising in the field of AI and ML, along with big data.

Sign language is a type of communication that uses hand signs and hand movements to communicate with others. The main motivation behind this project is that the application of this model can help hearing-challenged, speech-challenged, and visually-challenged humans to interact with a machine and work on them. It is said that one out of every ten people is hearing challenged; hence this project can help a large percentage of the population in communication. Our goal is to develop an ML product that detects hand signs (sign language) and acts as a means of communication between the human and the machine.

Conclusion

The main objective of this project was to create a ML model to read, understand and predict images. This landmark has been accomplished successfully with high prediction accuracy and with less accuracy error. The model has performed successfully in predicting sign language images. More importantly, these images are hand postures of sign language images used by hearing-challenged humans; hence this model has an important application that can be used as an interface between the human and the machine. This basic model can have large applications with more additional capabilities added in the near future. Furthermore, an interactive web interface hosting this CNN predictive model has been created, and the same has been deployed into the Microsoft Azure platform successfully.

Future Works

As an extension of this project, we could build an interactive Graphical User Interface (GUI) on this model that can read sign language in real-time using webcams from laptops or Desktops for communication. The VGG19 Convolution Neural Network (CNN) will be a good place to start exploring since this project already covers DenseNet201 and the Sequential model. The same model can be used to develop and made into mobile applications.
